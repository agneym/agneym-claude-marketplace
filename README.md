# Agney's Claude Marketplace

Official marketplace for my plugin collection. Currenly contains an imagen plugin.

Refer to [readme](./plugins/README.md) for plugin information.

## Quick Start

### Standard Installation

Run Claude and add the marketplace:

```bash
/plugin marketplace add https://github.com/agneym/agneym-claude-marketplace
```

Then install the plugin:

```bash
/plugin install explore-with-illustrations
```
